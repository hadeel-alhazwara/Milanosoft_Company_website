# Milano Soft Website
## Author: Hadeel Al-Hazwara

---

## 📌 Project Description
This repository contains the official website for Milano Soft, a software company. The website showcases company information, services, projects, and contact details.

---

## 🗂 Project Structure
- `index.html` → Main homepage
- `about.html` → About the company (optional)
- `contact.html` → Contact page (optional)
- `css/` → Stylesheets
- `js/` → JavaScript files
- `images/` → Logo and other images

---

## ▶ How to View
Simply open `index.html` in any web browser.  
For live hosting, GitHub Pages can be enabled in the repository settings.
